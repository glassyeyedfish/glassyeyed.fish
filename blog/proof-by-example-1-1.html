An Unnessecarily Detailed Math Proof

In this series of articles the goal is simple. Teach you how to write math proofs using only a series of examples. This is because I think understanding examples and then practicinig exercieses are the best way to learn math (and anything really).

Let's not was time and get into the first example.

Example 1

Problem:

Prove that: For all \(n \in \ZZ\), if \(n\) is even, then \(n^2\) is even.

Solution:

  1. Fix \(n \in \ZZ\).
  2. Assume \(n\) is even.
  3. By definition (2), there exists \(k \in \ZZ\) such that \(n = 2k\).
  4. By the reflexive property of equality, \(n^2 = n^2\).
  5. By substitution (3, 4), \(n^2 = (2k)^2\).
  6. By algebra, \((2k)^2 = 2(2k^2)\).
  7. By the transitive property of equality (5, 6), \(n^2 = 2(2k^2)\)
  8. \(2 \in \ZZ\).
  9. By closure (3, 8), \(2k^2 \in \ZZ\).
  10. I choose \( l = 2k^2 \).
  11. By substitution (9, 10), \( l \in \ZZ \).
  12. By substitution (7, 10), \( n^2 = 2l \).
  13. By existential introuction (11, 12), there exists \( l \in \ZZ \) such that \( n^2 = 2l \).
  14. By definition (13), \(n^2\) is even.
  15. By direct proof (2-14), if \(n\) is even, then \(n^2\) is even.
  16. By universal generalization (1, 16), for all \(n \in \NN\), if \(n\) is even, then \(n^2\) is even.

First thing to notice. Normally, in textbooks, proofs are not written this way. Typically, they are a solid paragraph of text, and they don't have this many details. A more typical version of this proof might look like this.

Assume \(n\) is even. Thus, \(n = 2k\) for some \(k \in \ZZ\). Since \(n^2 = 4k^2\), it's clear that \(n^2\) is also even as required.

The reason I have chosen to write these proofs as a list which contains this much detail, is because I want to make it clear that every proof can be broken down into a series of simple steps. It will easy to check the following:

Every statement in a proof is either:
  1. Derived: meaning it follows from zero or more previous statements.
  2. Declared: not derived, but still nessecary to construct a correct proof.
So, when we have a list of statements, where the last statement is the statement we originally wanted to prove, and each statement was correctly derived or declared, then we know we have a correct proof!

This is SUPER IMPORTANT! It's why is outlined it in red! We will now spend the rest of this article checking each step of this proof in reverse, and explaning why each step is important to the complete and correct proof.

We will start from the last line, and work backwards.

Line 11

  1. Fix \(n \in \ZZ\).
  2. Assume \(n\) is even.
  3. By definition, this means that there exists \(k \in \ZZ\) such that \(n = 2k\).
  4. Using some algebra, we see that \( n^2 = (2k)^2 = 2(2k^2) \).
  5. I'll choose \( l = 2k^2 \).
  6. Thus, by substitution, \( n^2 = 2l \).
  7. By closure, we see that \( l \in \ZZ \).
  8. By existential introuction, we know that there exists \( l \in \ZZ \) such that \( n^2 = 2l \).
  9. By definition, that means \(n^2\) is even.
  10. By direct proof we obtain that if \(n\) is even, then \(n^2\) is even.
  11. By universal generalization, we conclude that for all \(n \in \NN\), if \(n\) is even, then \(n^2\) is even.

Here, line 11 is an example of a derived statement. This means, as we just saw, that it follow from zero or more previous statements. There is also a justification for why line 11 follows from line 1 and line 10.

In this case, we are using a rule of inference called "universal generalization". This rule works as follows:

To use universal generalization, we need to:
  1. Fix some variable \(x\) with a certain condition,
  2. derive some statement \(P(x)\),
  3. and all other assumptions about \(x\) have been discharged.
and then we can conclude that:
  • For all \(x\) with the condition, \(P(x)\) is true.

To prove step 11, on line 1 we fix the variable "\(n\)" with the condition that "\(n \in \ZZ\)", on line 10 we derive the statement "if \(n\) is even, then \(n^2\) is even", and all other assumptions about "\(n\)" have been discharged, we can conclude that step 11 is a correct step in the proof.

You might have noticed that on line 2, we actually made a futher assumption about \(n\). However, this assumption is discharged on line 10 when we use direct proof. We will discuss this when we discuss line 10.

Line 10

  1. Fix \(n \in \ZZ\).
  2. Assume \(n\) is even.
  3. By definition, this means that there exists \(k \in \ZZ\) such that \(n = 2k\).
  4. Using some algebra, we see that \( n^2 = (2k)^2 = 2(2k^2) \).
  5. I'll choose \( l = 2k^2 \).
  6. Thus, by substitution, \( n^2 = 2l \).
  7. By closure, we see that \( l \in \ZZ \).
  8. By existential introuction, we know that there exists \( l \in \ZZ \) such that \( n^2 = 2l \).
  9. By definition, that means \(n^2\) is even.
  10. By direct proof we obtain that if \(n\) is even, then \(n^2\) is even.
  11. By universal generalization, we conclude that for all \(n \in \NN\), if \(n\) is even, then \(n^2\) is even.

Here, to prove line 10, we aren't just using line 2 and line 9. We are actually connsidering the entire arguement from line 2 up to line 9. To see why this is the case, let's quickly see how direct proof works.

To use direct proof, we need to:
  1. Start a sub-proof by assuming some statement \(P\)
  2. Prove some statement \(Q\) inside the sub-proof.
  3. Dischange the assumption to end the sub-proof.
and then we can conclude:
  • If \(P\), then \(Q\).

Here, starting from line 2 we are assuming that \(n\) is even. Then, after line 9 with dischange that assumption. This allows us to conclude the "if, then" statement on line 10.
