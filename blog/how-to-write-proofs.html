How to Write Math Proofs, Part I: Propositional Logic

In university level maths, it can be hard to know how proofs are written. The goal of this set of blog posts is to teach you how to write correct math proofs, and more than that, how to know when the proofs you write are actually correct.

The first step in this process, is something I will call...

Propositional Logic

The idea of propositional logic is to look at how different propositions (or statements) can be combined to make more complex statements. But first, we have to know exactly what a proposition is.

Definition: A proposition is a sentence that we can decide is either true or false.

Some examples of propositions in regular English are:

  • The sky is blue.
  • The moon is made of cheese.
  • Every number is either even or odd.

Some examples of sentences that are not prositions are:

  • What time is it?
  • Go move those boxes to the other room!
  • This statement is false.
The last line in that list of examples may seem like a statement, but the problem arises when you try to decide if it's true or false. I encourage you to try and decide if "this statement is false" is true or false, and you should hopefully then understand why it's not a statement.
